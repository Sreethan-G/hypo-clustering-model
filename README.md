# Hypothyroid Patient Clustering

Code for "Identifying phenotypes of hypothyroid patients treated with
levothyroxine using unsupervised machine learning."

## Data
Download allhypo.data from the UCI Thyroid Disease dataset:
https://archive.ics.uci.edu/dataset/102/thyroid+disease

Place the file in data/allhypo.data before running.

## Setup
pip install -r requirements.txt

## Usage
python train_model.py
